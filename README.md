# MoooseMini
55mm大玉搭載の自作マウス"MoooseMini"のZMK-Configリポジトリです。  
![](./Readme_image/MoooseMini01.png)

## 特徴
・ZMK Firmware採用で有線/無線可能  
・ZMK Studio対応  
・ロータリーエンコーダ対応  
・最大8キー(MX互換)+1タクトスイッチ  
・ロック機能付きトラボ筐体
